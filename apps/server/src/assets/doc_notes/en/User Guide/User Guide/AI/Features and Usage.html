AI Features and Usage Guide

Overview

This guide explains how to effectively use AI features in Trilium Notes to enhance your knowledge management workflow.

Chat with Notes

Getting Started

The Chat with Notes feature provides an interactive AI assistant that has access to your knowledge base.

Accessing the Chat Interface

  1. Via Launcher Button: Click the "Chat with Notes" button in the left sidebar
  2. Via Keyboard Shortcut: Press Ctrl+Shift+C (customizable)
  3. Via Command Palette: Press Ctrl+P and type "chat"

Basic Chat Operations

Asking Questions

The AI can answer questions based on your notes:

User: What are the main points from my meeting notes this week?
AI: Based on your meeting notes from this week, here are the main points:
- Monday: Project timeline discussed, deadline set for March 30
- Wednesday: Budget approved for Q2
- Friday: New team member onboarding planned

Summarizing Content

User: Summarize my research notes on machine learning
AI: Your machine learning research covers:
1. Supervised learning algorithms (Random Forests, SVM)
2. Neural network architectures
3. Data preprocessing techniques
4. Model evaluation metrics

AI-Powered Tools

Available Tools

The AI can execute various tools to interact with your notes:

Tool Function Example Usage
search_notes Semantic search across notes "Find all notes about project management"
keyword_search Exact keyword matching "Search for notes containing 'API documentation'"
read_note Access full note content "Show me the content of my budget proposal"
create_note Generate new notes "Create a note summarizing this discussion"
update_note Modify existing notes "Add these action items to my todo list"
manage_attributes Handle note metadata "Add #important label to my proposal"
calendar_integration Work with date notes "What's on my calendar tomorrow?"

Semantic Search

How It Works

Semantic search uses AI embeddings to find conceptually related content, even when exact keywords don't match.

Benefits

  • Find related concepts without exact keywords
  • Discover unexpected connections
  • More intuitive search results
  • Better context understanding

Example Searches

Query: "improving productivity"
Finds: Time management, GTD methodology, focus strategies, tool optimization

Query: "customer satisfaction"
Finds: User feedback, NPS scores, support tickets, product reviews

Content Generation

Note Creation

AI can help create various types of notes:

  • Meeting Summaries: "Create a summary of today's standup discussion"
  • Research Outlines: "Generate an outline for my thesis on climate change"
  • Documentation: "Create API documentation from these code snippets"
  • Reports: "Generate a weekly progress report from my task notes"

Content Enhancement

Improve existing notes with AI assistance:

  • Expansion: "Expand the bullet points in my ideas note"
  • Clarification: "Rewrite this technical explanation for beginners"
  • Formatting: "Convert my rough notes into a structured document"
  • Translation: "Translate this note to Spanish"

Best Practices

Effective Prompting

Instead of... Try...
"Tell me about my notes" "Summarize my project management notes from Q1"
"What should I do?" "Based on my project timeline, what tasks should I prioritize?"
"Give me information" "Create a bullet-point summary of my research on AI"

Managing Conversations

  • Start fresh: Begin new chats for different topics
  • Be specific: Reference note titles when possible
  • Iterate: Use follow-up questions to refine responses
  • Provide context: Include relevant background information

Performance Tips

  • For speed: Use simpler models for basic tasks
  • For quality: Provide examples of desired output
  • For cost: Enable caching and batch similar queries
  • For relevance: Keep your notes well-organized and labeled

Privacy and Control

Excluding Notes from AI

To prevent specific notes from being accessed by AI:

  1. Add the #excludeFromAI label to sensitive notes
  2. These notes won't be included in searches or context
  3. Perfect for personal or confidential information

Data Usage

  • Only explicitly requested notes are sent to AI
  • No automatic uploads or background processing
  • You control what data the AI can access

Troubleshooting Common Issues

AI Not Finding Relevant Notes

Solution: Ensure embeddings are generated. Go to Settings ‚Üí AI/LLM and click "Recreate All Embeddings".

Tools Not Executing

Solution: Verify tool calling is enabled in your AI settings and your provider supports function calling.

Slow Responses

Solution: Try using a faster model (GPT-3.5-turbo, Claude Haiku) or reduce the context size in settings.
