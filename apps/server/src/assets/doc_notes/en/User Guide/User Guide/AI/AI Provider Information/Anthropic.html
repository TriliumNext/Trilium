Anthropic Configuration Guide

Overview

Anthropic provides access to the Claude 3 family of models, known for their strong analytical capabilities, safety features, and large context windows. This guide will help you configure Anthropic as your AI provider in Trilium Notes.

Getting Started

Step 1: Create an Anthropic Account

  1. Visit Anthropic Console
  2. Sign up with your email address
  3. Verify your email
  4. Complete account setup and billing information

Step 2: Generate an API Key

  1. Log into the Anthropic Console
  2. Navigate to API Keys section
  3. Click "Create Key"
  4. Name your key (e.g., "Trilium Integration")
  5. Important: Copy and save the key immediately
  6. Store securely - the key won't be shown again

Security Notice

Your API key provides full access to your Anthropic account. Never share it publicly or commit it to version control.

Step 3: Configure in Trilium

  1. Open Trilium Notes
  2. Go to Options ’ AI/LLM
  3. Enable AI features
  4. Select Anthropic from the provider dropdown
  5. Enter your configuration:
    • API Key: Your Anthropic API key (sk-ant-...)
    • Base URL: https://api.anthropic.com (default)
    • Default Model: Choose from available Claude models
  6. Click Test Connection to verify

Configuration Options

Model Parameters

  • Temperature (0.0-1.0): Controls response randomness
    • 0.0-0.3: Precise, consistent responses
    • 0.4-0.7: Balanced creativity and accuracy
    • 0.8-1.0: Creative, varied outputs
  • Max Tokens: Maximum response length
    • Default: 4096 tokens
    • Maximum: Model-dependent (up to 200K)

Best Practices

Prompt Engineering for Claude

  • Be Direct: Claude responds well to clear, direct instructions
  • Use XML Tags: Structure complex prompts with XML-like tags for better organization
  • Provide Examples: Claude excels at following patterns from examples
  • Think Step-by-Step: For complex tasks, ask Claude to reason through steps
