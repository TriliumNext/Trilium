# Gallery
## Gallery View

The Gallery View displays visual notes (images, canvases, mermaid diagrams, and mind maps) as a grid of thumbnails, making it ideal for browsing and managing collections of visual content such as photo albums, artwork, or diagram collections.

## Interaction

*   **Click** on a thumbnail to open the note.
*   **Ctrl+Click** (or **Cmd+Click** on Mac) to add or remove individual notes from the selection.
*   **Shift+Click** to select a range of notes between the last selected note and the clicked note.
*   **Right-click** to access the context menu with options to open in a new window, rename, share, or delete notes.

### Selection mode

When notes are selected, the toolbar displays:

*   The count of selected items.
*   A **Clear Selection** button to deselect all.
*   A **Delete Selected** button to delete all selected notes at once.

Use **Select All** to quickly select all items in the gallery.

## Toolbar actions

The toolbar at the top provides quick actions:

*   **Back to Parent** - When viewing a nested gallery, navigate back to the parent gallery.
*   **New Gallery** - Create a nested gallery folder to organize content.
*   **Upload** - Upload multiple images directly into the gallery with progress indication.
*   **Select All** / **Clear Selection** - Manage multi-selection.
*   **Delete Selected** - Remove selected notes (available when items are selected).

The toolbar also displays counts of galleries (folders) and images in the current view.

## Sharing

Shared notes display a share badge on their thumbnail.

*   **Click the share badge** to copy the share URL to the clipboard.
*   **Right-click** and use the context menu to toggle sharing on or off for individual notes.

## Context menu

Right-clicking on a note (or selection) provides:

*   **Open note in new window** - Opens the note in a separate window.
*   **Rename note** - Change the note's title (single selection only).
*   **Share / Unshare** - Toggle public sharing (single selection only).
*   **Delete** - Remove the note(s) from the gallery.

When multiple notes are selected, bulk delete is available.
